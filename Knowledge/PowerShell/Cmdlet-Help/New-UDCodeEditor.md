# New-UDCodeEditor

## SYNOPSIS
Creates a new Monaco code editor control.

## SYNTAX

```
New-UDCodeEditor
```

## DESCRIPTION
Creates a new Monaco code editor control.

## EXAMPLES

### BEISPIEL 1
```
New-UDCodeEditor -Code 'Get-Process' -Theme 'vs-dark' -Language 'powershell' -Readonly
```

Creates a readonly code editor with PowerShell script.
