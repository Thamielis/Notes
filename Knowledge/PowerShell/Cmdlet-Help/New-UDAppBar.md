# New-UDAppBar

## SYNOPSIS
Creates an AppBar.

## SYNTAX

```
New-UDAppBar
```

## DESCRIPTION
Creates an AppBar.
This can be used to replace the built-in AppBar.

## EXAMPLES

### BEISPIEL 1
```
Creates a new AppBar that is relative to other components.
```

New-UDAppBar -Children { New-UDTypography -Text 'Hello' -Paragraph } -Position relative
