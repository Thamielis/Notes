New-Logger
----------

### Synopsis
Creates new logger

---

### Description

Creates instance of serilog logger configuration

---

### Examples
> EXAMPLE 1

```PowerShell
New-Logger
```

---

### Inputs
None. You cannot pipe objects to New-Logger

---

### Outputs
* Instance of LoggerConfiguration object allowing method chaining

---

### Syntax
```PowerShell
New-Logger [<CommonParameters>]
```
