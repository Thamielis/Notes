Stop-GitHubLogGroup
-------------------

### Synopsis
Stops the current log group in GitHub Actions

---

### Description

---

### Examples
> EXAMPLE 1

```PowerShell
Stop-LogGroup
Starts a new log group named 'MyGroup'
```
