New-NcSystemHypervisorHostConfig
--------------------------------

### Synopsis
Create the hypervisor host config.

---

### Description

Create the hypervisor host config.

---

### Examples
> Example 1

```PowerShell
New-NcSystemHypervisorHostConfig -Node sapanj-vsim3 -AddressIpv4 10.140.111.44
Create the hypervisor host config.

AddressIpv4       : 10.140.111.44
NcController      : 10.140.111.42
Node              : sapanj-vsim3
PortIpv4          : 161
PortIpv4Specified : True

```
