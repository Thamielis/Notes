Start-NcSnaplockLegalHoldDumpFiles
----------------------------------

### Synopsis
Dump the list of files under legal hold for a given volume and litigation to a user specified output path. The progress of the operation can be monitored using snaplock-legal-hold-get-iter api.

---

### Description

Dump the list of files under legal hold for a given volume and litigation to a user specified output path. The progress of the operation can be monitored using snaplock-legal-hold-get-iter api.

---
