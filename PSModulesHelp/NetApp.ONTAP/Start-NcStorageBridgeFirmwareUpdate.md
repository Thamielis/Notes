Start-NcStorageBridgeFirmwareUpdate
-----------------------------------

### Synopsis
Download firmware to the specified bridge. The bridge needs to be rebooted for the firmware to be updated successfully.

---

### Description

Download firmware to the specified bridge. The bridge needs to be rebooted for the firmware to be updated successfully.

---

### Parameters
#### **Bridge**
The bridge to be updated.

|Type      |Required|Position|PipelineInput        |
|----------|--------|--------|---------------------|
|`[String]`|true    |1       |true (ByPropertyName)|

#### **Uri**
The URI location of the firmware file.

|Type      |Required|Position|PipelineInput        |
|----------|--------|--------|---------------------|
|`[String]`|true    |2       |true (ByPropertyName)|

#### **Controller**

|Type              |Required|Position|PipelineInput                 |Aliases                          |
|------------------|--------|--------|------------------------------|---------------------------------|
|`[NcController[]]`|false   |named   |true (ByValue, ByPropertyName)|Filer<br/>Server<br/>NcController|

#### **InformationAction**

* SilentlyContinue
* Stop
* Continue
* Inquire
* Ignore
* Suspend

|Type                |Required|Position|PipelineInput|Aliases|
|--------------------|--------|--------|-------------|-------|
