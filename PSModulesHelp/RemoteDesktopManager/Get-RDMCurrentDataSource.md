Get-RDMCurrentDataSource
------------------------

### Synopsis
Retrieve the active data source.

---

### Description

Retrieve the active data source.

---

### Examples
> EXAMPLE

```PowerShell
PS C:\> $list = Get-RDMCurrentDataSource
The current active data sources.
```
