Get-Arguments
-------------

### Synopsis
Uses command-line arguments

---

### Description

---

### Parameters
#### **FirstNamedArgument**
The first named argument

|Type      |Required|Position|PipelineInput|
|----------|--------|--------|-------------|
|`[Object]`|false   |1       |false        |

#### **SecondNamedArgument**
The second named argument

|Type     |Required|Position|PipelineInput|
|---------|--------|--------|-------------|
|`[Int32]`|false   |2       |false        |

---

### Syntax
```PowerShell
Get-Arguments [[-FirstNamedArgument] <Object>] [[-SecondNamedArgument] <Int32>] [<CommonParameters>]
```
