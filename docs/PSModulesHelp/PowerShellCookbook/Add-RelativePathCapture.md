Add-RelativePathCapture
-----------------------

### Synopsis
Adds a new CommandNotFound handler that captures relative path
navigation without having to explicitly call 'Set-Location'

---

### Description

---

### Examples
> EXAMPLE 1

```PowerShell
..
PS C:\Users\Lee>...
PS C:\>
```

---

### Syntax
```PowerShell
Add-RelativePathCapture [<CommonParameters>]
```
