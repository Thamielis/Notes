Get-RDMPrivateSessionAttachment

### Synopsis
Get session attachments

---

### Description

Get session attachments

---

### Parameters

|Type            |Required|Position|PipelineInput|
|----------------|--------|--------|-------------|
|`[PSConnection]`|true    |1       |false        |
