Start-TimeLog
-------------

### Synopsis
Starts a new stopwatch for logging time.

---

### Description

This function starts a new stopwatch that can be used for logging time durations.

---

### Examples
> EXAMPLE 1

```PowerShell
Start-TimeLog
Starts a new stopwatch for logging time.
```

---

### Syntax
```PowerShell
Start-TimeLog [<CommonParameters>]
```
