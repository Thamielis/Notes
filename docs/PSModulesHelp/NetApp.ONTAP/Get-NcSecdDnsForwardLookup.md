Get-NcSecdDnsForwardLookup

### Description

DNS Forward Lookup for a Hostname

|`[String]`|false   |named   |true (ByPropertyName)|

|----------|--------|--------|---------------------|
|`[String]`|true    |2       |true (ByPropertyName)|
