Set-NcVolumeAutosizeAsync
-------------------------

### Synopsis
Given the name of a volume, set the autosize settings.

---

### Description

Given the name of a volume, set the autosize settings.

---

### Examples
> Example 1

```PowerShell
Set-NcVolumeAutosizeAsync -Volume vol1 -MaximumSize 1g
Sets the MaximumSize to 1G
```

---

### Parameters
#### **Volume**
The name of the flexible volume for which we want to set autosize.

|Type      |Required|Position|PipelineInput        |
|----------|--------|--------|---------------------|
|`[String]`|true    |1       |true (ByPropertyName)|

|---------|--------|--------|---------------------|
|`[Int32]`|false   |named   |true (ByPropertyName)|

|Type      |Required|Position|PipelineInput|Aliases|
