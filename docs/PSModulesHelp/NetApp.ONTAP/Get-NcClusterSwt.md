Get-NcClusterSwt
----------------

### Synopsis
Iterate over a list of Cluster Switch Information objects.

---

### Description

Iterate over a list of Cluster Switch Information objects.

---

### Examples
> Example 1

```PowerShell
Get-NcClusterSwt -Node sti32-vsim-ucs480e
List all the cluster swt info objects on the node sti32-vsim-ucs480e
```
