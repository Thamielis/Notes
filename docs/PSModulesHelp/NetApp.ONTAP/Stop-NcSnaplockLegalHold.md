Stop-NcSnaplockLegalHold
------------------------

### Synopsis
This API is used to abort an ongoing legal-hold operation. This API only aborts legal-hold operations that have started and not  finished. The operation ID of the legal-hold operation must be specified.

---

### Description

This API is used to abort an ongoing legal-hold operation. This API only aborts legal-hold operations that have started and not  finished. The operation ID of the legal-hold operation must be specified.
