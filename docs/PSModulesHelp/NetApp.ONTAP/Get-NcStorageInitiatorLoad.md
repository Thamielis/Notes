Get-NcStorageInitiatorLoad
--------------------------

### Synopsis
Gets disk I/O rates for a given fibre channel initiator port or for all initiator ports.

---

### Description

Gets disk I/O rates for a given fibre channel initiator port or for all initiator ports. The disk can refer to an array LUN, actual disk, or solid state device.

---

### Examples
> Example 1

```PowerShell
Get-NcStorageInitiatorLoad
Gets disk I/O rates for all initiator ports.
```

---

### Parameters
#### **Port**
Port to show load on, e.g. 0a.

|Type      |Required|Position|PipelineInput        |
|----------|--------|--------|---------------------|
|`[String]`|false   |1       |true (ByPropertyName)|

#### **Node**
Obtain disk I/O rates for initiator ports on the specified node.

|Type      |Required|Position|PipelineInput        |
|----------|--------|--------|---------------------|
|`[String]`|false   |2       |true (ByPropertyName)|

#### **Controller**
A clustered Data ONTAP controller to receive this cmdlet, as embodied by an NcController object.  This parameter is returned by the Connect-NcController cmdlet.  If not specified, the value in the global variable CurrentNcController is used.  In the latter case, if CurrentNcController contains a collection of NcController objects, this cmdlet is invoked on each controller in succession.

|Type              |Required|Position|PipelineInput                 |Aliases                          |
|------------------|--------|--------|------------------------------|---------------------------------|
|`[NcController[]]`|false   |named   |true (ByValue, ByPropertyName)|Filer<br/>Server<br/>NcController|

#### **InformationAction**
