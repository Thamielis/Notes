Get-OperatingSystemShortName
----------------------------

### Synopsis
Returns the Platform name.

---

### Description

Gets whether the platform is Windows, Linux or MacOS.

---

### Examples
> EXAMPLE 1

```PowerShell
Get-OperatingSystemShortName # no Parameter needed
```
