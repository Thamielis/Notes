Show-OAIBilling
---------------

### Synopsis
Opens the OpenAI platform billing overview page.

---

### Description

The Show-OAIBilling function opens the OpenAI platform billing overview page in the default web browser.

---

### Examples
> EXAMPLE 1

```PowerShell
Show-OAIBilling
Opens the OpenAI platform billing overview page.
```

---

### Syntax
```PowerShell
Show-OAIBilling [<CommonParameters>]
```
