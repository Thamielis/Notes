Show-OAIPlaygroundWebPage
-------------------------

### Synopsis
Opens the OpenAI Playground Web Page.

---

### Description

This function opens the OpenAI Playground Web Page in the default web browser.

---

### Examples
> EXAMPLE 1

```PowerShell
Show-OAIPlaygroundWebPage
```

---

### Syntax
```PowerShell
Show-OAIPlaygroundWebPage [<CommonParameters>]
```
