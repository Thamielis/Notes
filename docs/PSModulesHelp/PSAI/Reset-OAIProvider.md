Reset-OAIProvider
-----------------

### Synopsis
Resets the OpenAI provider.

---

### Description

The Reset-OAIProvider function is used to reset the OpenAI provider.

---

### Examples
> EXAMPLE 1

```PowerShell
Reset-OAIProvider
Resets the OpenAI provider.
```

---

### Syntax
```PowerShell
Reset-OAIProvider [<CommonParameters>]
```
