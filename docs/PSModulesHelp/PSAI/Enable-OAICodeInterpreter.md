Enable-OAICodeInterpreter
-------------------------

### Synopsis
Enables the OAI code interpreter.

---

### Description

The Enable-OAICodeInterpreter function enables the OAI (OpenAI) code interpreter.

---

### Examples
> EXAMPLE 1

```PowerShell
Enable-OAICodeInterpreter
```

---

### Syntax
```PowerShell
Enable-OAICodeInterpreter [<CommonParameters>]
```
