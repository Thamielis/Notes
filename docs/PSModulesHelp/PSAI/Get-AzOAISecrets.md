Get-AzOAISecrets
----------------

### Synopsis
Retrieves the AzOAI secrets.

---

### Description

The Get-AzOAISecrets function retrieves the AzOAI secrets.

---

### Examples
> EXAMPLE 1

```PowerShell
Get-AzOAISecrets
This example demonstrates how to use the Get-AzOAISecrets function to retrieve the AzOAI secrets.
```
